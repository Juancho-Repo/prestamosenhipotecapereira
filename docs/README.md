## Despliegue rápido en GitHub Pages

1. Sube la carpeta `landing/` a un repositorio de GitHub
2. Ve a Settings → Pages
3. Selecciona rama `main` y carpeta `/root`
4. Tu landing estará en `https://tuusuario.github.io/landing`
