# 📁 src/assets/

Recursos estáticos del proyecto.

```
src/assets/
├── images/   → Imágenes de uso general (logos, ilustraciones, banners)
└── icons/    → Íconos SVG (si no usas emoji)
```

> Las fotos de clientes van en /images/clientes/ (raíz del proyecto).
> Las fuentes van en /public/fonts/ si las auto-hostas.
